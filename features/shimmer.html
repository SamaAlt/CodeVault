@keyframes shimmer {
    0% {
        background-position: -400px 0;
    }
    100% {
        background-position: 400px 0;
    }
}
// No JavaScript required for basic shimmer effect

// Optional: Add this to apply shimmer to multiple elements dynamically
document.querySelectorAll('.shimmer').forEach(el => {
    el.classList.add('shimmer-effect');
});

Usage Notes

  • Great for loading states and placeholders
  • Adjust gradient colors to match your design
  • Modify animation duration for different effects
  • Works best on elements with defined dimensions
